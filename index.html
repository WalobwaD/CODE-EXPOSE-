Instructions

Start

Click the start button only when it's the first day you're using the app and want to track your coding days and time spent while coding. The button will forever be disabled after the first click.

Pause

Click the pause button after you're done for the day and want to stop coding and you want to resume later. The Pause button will be activated only when the time is running and gets deactivated when the timer stops running.

Resume

Click the resume button when you are back to let the timer pick up from where you left. The resume button will only get activated after clicking the pause button.
